# DailyFocus

DailyFocus is a simple command-line utility that helps you stay organized and focused by tracking your top three priorities for the day. Easily view, update, or clear your priorities anytime. This utility is not only useful for managing daily tasks but also provides a starting point for learning Python programming.

## Usage

To use this utility, open a terminal and navigate to the folder where you saved `dailyfocus.py`. Then, run the script using the following commands:

### Set your top 3 priorities for the day:
```bash
python dailyfocus.py set "Priority 1" "Priority 2" "Priority 3"

### View your priorities:
python dailyfocus.py show

### Clear your priorities:
python dailyfocus.py clear

## Alias Setup

To make it even easier to use, you can add an alias to your shell configuration file (e.g., `.bashrc`, `.zshrc`) that points to the script. That way, you can simply type `dailyfocus` followed by the desired command.
